【このフォルダについて】
2020年ごろに画面からDBまで疎通させた処理を作りたくて
要件定義、画面仕様、機能一覧、テーブル設計、コード設計を作成しました。

要件のベースは、普段使っている家計簿です。
家計簿はExcelで15年くらい改良し続けており、要件も機能も把握しています。
MVCモデルに落とし込んだ場合のドキュメントとなります。
　→　当時の案件でSpring使っていたので、SpringMVCで実装しようと考えていました。

Gitにはありませんが、ドキュメント作成の他に以下を行いました。
　・サーバー（非公開）は仮想マシン上にCentOSをインストールして構築しました。
　・PostgreSQLサーバーの構築、データベースの作成
　　　→　テーブル定義はExcelで行い、VBAを使ってCREATE文を出力するツールを作りました。
　・Webサーバーの構築、一部画面をJavaScriptで作成（サーバー処理との疎通なし）


コード設計までは、ほぼ完了させたような記憶はありますが、
プライベートの時間でやるには現実的な規模ではなくなってしまったので、製造はやめました。
