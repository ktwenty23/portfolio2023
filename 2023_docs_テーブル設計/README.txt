【背景】
データベースの作成を簡略化するために、Excelで定義したテーブル情報をVBAを使ってSQL文を出力させるツール（対象：PostgreSQL）を作成しました。

　・DDL生成ツール.xlsm
　　仕様：テーブル定義.xlsx からテーブルごとに.sqlファイル（CREATE文）を出力する。
　　　　　合わせて出力したCREATE文を一括で実行できるように.shファイルを出力する。
　　使い方：DBサーバーにログインして、出力されたファイルを配置し、shを実行する。

　・SQL文出力ツール.xlsm
　　仕様：*_DATA.xlsx から.sqlファイル（INSERT文）を出力する。
　　使い方：DBViewerなどにINSERT文をコピペして使う。
